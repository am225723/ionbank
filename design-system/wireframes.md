## User Journey Flow

1. **Landing** → Homepage hero section
2. **Learn** → Features and how it works
3. **Try** → SOP Generator demo/trial
4. **Create** → Full SOP Generator access
5. **Export** → Download/share documents
6. **Manage** → Document library and settings
