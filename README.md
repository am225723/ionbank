## 📦 Quick Start Guide

### Prerequisites
- Node.js 16.x or higher
- npm 7.x or higher

### Installation Steps

```bash
# Clone the repository
git clone https://github.com/am225723/ionbank.git
cd ionbank/sop-generator

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:5173 in your browser
```
